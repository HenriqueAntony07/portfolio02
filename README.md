# Henrique Antony — Portfolio

  ## Como rodar localmente

  ```bash
  npm install
  npm run dev
  ```

  Abre em http://localhost:3000

  ## Como buildar para produção

  ```bash
  npm run build
  ```

  Os arquivos gerados ficam em `out/` — é só subir na hospedagem.

  ## Adicionar imagens dos projetos

  Coloque as imagens em `public/images/`:
  - `public/images/pulse.png`
  - `public/images/domum.png`
  - `public/images/makemais.png`

  ## Stack
  - Next.js 15
  - React 19
  - Tailwind CSS v4
  - Framer Motion
  - next-themes (dark/light mode)
  - TypeScript
